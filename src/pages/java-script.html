Nauka javaScript'u

javaScript

javaScript uruchamia się zaraz po załadowaniu strony. Na pierwszy ogień poznaliśmy z czego składa się:

  • Wartości - dla JS są tym czym liczby dla matematyki. Nie tworzy ich, raczej sie do nich nawiązujemy.
  • Operacje porównania
  • Literały
  • Zmienne
  • Obiekty
  • Funkcje

Pliki zawierające kod JS mają rozszerzenie .js ale podobnie jak style - mają swój znacznik <script></script>;

Poznaliśmy sposób w JS'ie zapisu:

Zmiennej:

const name='Paweł'; - zmienna name: Paweł
const age=34; - zmienna age o wartości 34

Komendę wywołującą zapis w konsoli przeglądarki:

console.log(name) - zwróci w konsoli wartość zmiennej name czyli napis Paweł
console.log(age) - zwróci w konsoli wartość zmiennej age czyli 34

Komendę alert, która wyświeli w osobnym okieny wyświetli zdefiniowany komunikat:
alert('Hejka")- wyświeli komunikat Hejka
Specjalny sposób zapisu literału, czyli tekstu, w kórym umieszczone są zmienne:
`Cześć :) Nazywam się ${name} i mam ${age} lat`; gdzie ${zmienna} to zapis odwołujący się do zmiennej.

Z rzeczy do zapamiętania:

  • Java to nie javaScript
  • javaScript implementuje rzeczy ze specyfikacju ECMA Script
  • javaScript to frontend'owe programowanie
