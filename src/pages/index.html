Wartwy strony internetowej

Zaczynamy od poznania poszczególnych warstw dokumentu. W jego skład wchodzi:

  • Warstwa 0: Tekst.
  • Warstwa 1: HTML.
  • Warstwa 2: CSS.
  • Warstwa 3: JavaScript

Poszczególne warstwy mają wpływ na siebie. Na tekst (czyli wszystko co się wyświetla w dokumencie, czyli dodatkowo obrazy, linki, tabele itd.) mają wpływ inne warstwy. HTML nadaje strukturę stronie. Stanowi jej szkielet. CSS odpowiada za wygląd. A w JavaScript dodaje interaktywność do strony. HTML wpływa bezpośrednio tylko na tekst, za to CSS już może wywierać wpływ na kod HTML i tekst. JavaScript może wywołać najwięcej zamieszania. Wpływać już może na: tekst, kod HTML i CSS.

Nadawanie nazw elementom

Jest to jedna z najtrudniejszych czynności do wykonania. Żeby wprowadzić porządek najlepiej stosować kilka prostych zasad:

  • nazwy bez polskich znaków,
  • nie używamy spacji
  • najlepiej używać angielskich słów opisujących dany element,
  • trzymamy się jednego już istniejącego w dokumencie schematu.

Dla wprowadzenia czytelności danej kilku wyrazowej nazwy zapisanej bez spacji, można użyć jedne kilku istniejących konwencji zapisu:

camelCase
pierwsze słowo z małej litery, a kolejne wyrazy bez spacji zapisane są z dużej,
kebab-case
wszystkie wyrazy zapisujemy z małej litery, a oddzielamy je od siebie za pomocą myślnika "-",
snake_case
wszystkie słowa zapisujemy z małej litery i oddzielamy je "_",
PascalCase
wszystkie słowa zapisujemy z dużej litery, bez spacji między sobą.

Jest Git, a nawet gitHub :)

Git to program do zarządzania rozwijanym projektem oraz system kontroli wersji. Jest to narzędzie,dzięki któremu lepiej pracuje się w grupie i łatwo śledzi się zmiany, ewentualnie je wracać do wcześniejszej wersji. Umożliwia również rozdzielenie wersji i rozwijanie dwóch różnych wariantów projektu, by na koniec móc wybrać lepszą.

Program Git możemy uruchomić w terminalu poprzez polecenie: git. Musimy tylko pamiętać, żeby uruchomić Git'a w katalogu z rozwijanym projektem.

DOM’owe przedszkole

DOM czyli Document Object Model (Obiektowy model dokumentu).

  • Dom to złożony w przeglądarce HTML, już po wszystkich zmianach od JS.
  • JavaScript może modyfikować DOM.
  • Funkcja querySelector zwraca węzeł (node) z HTML’a, selktor zapisywany zgodnie z CSS.
  • Korzystając z BEM’a tworzymy dodatkową klasę z modyfikatorem –js (ułatwia identyfikację i usuwa potencjalne problem).
